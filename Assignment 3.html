HTML Tables

start tag for a Table

An HTML table is defined with the < table > tag.

start tag for a Table row

Each table row is defined with the < tr > tag.

start tag for a Table header

Each table header is defined with the < thead > tag.

start tag for a Table header (cell)

A table header is defined with the < th > tag.

start tag for a Table body

A table body is defined with the < tbody > tag.

start tag for a Table cell

A table cell is defined with the < td > tag.

start tag for a Table footer

A table footer is defined with the < tfoot > tag.
